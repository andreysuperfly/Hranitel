Хранитель — ограничивает время использования выбранных программ (Telegram, игры и др.) по расписанию.

Запустите установщик. Программа появится в трее. Откройте её по клику. Добавьте приложения (перетащите ярлык или exe), настройте время блокировки и включите её. По умолчанию блокировка с 00:00 до 08:00.

«Замок на 2 дня» — заблокирует все настройки на 2 дня.

Удаление: Панель управления → Программы → Хранитель
